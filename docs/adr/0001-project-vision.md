# ADR 0001 — Project Vision

## Status

Accepted

---

## Context

Many educational AI repositories focus on isolated implementations of transformers or language models.

NanoForge aims to teach the complete engineering process behind modern AI systems.

---

## Decision

NanoForge will implement every major component of an LLM pipeline from first principles.

The project emphasizes readability, testing, maintainability, and incremental development over raw performance.

---

## Consequences

### Positive

- Educational
- Easy to understand
- Modular
- Highly testable

### Negative

- More code to maintain
- Slower initial development
